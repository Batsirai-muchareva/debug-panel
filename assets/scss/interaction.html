Deep Dive

Behind the
Keyframes

01

Opacity

The most fundamental visibility control in CSS. Animating opacity alone triggers the GPU compositor layer, making it one of the most performant animations available.

02

Scale

Scale transforms affect perceived depth and hierarchy. Combined with transform-origin, they produce natural growth from any anchor point on the element.

03

Move

Translate moves elements without disturbing document flow — the other elements stay in place. It's why translate is preferred over top/left for animation.

04

Rotate

Continuous rotation with linear timing feels mechanical and intentional. Pair with ease-in-out for organic oscillation.

05

Skew

Skewing shears the coordinate system of the element. It's rarely used in production, which makes it feel fresh and editorial — perfect for expressive brand motion.

The Rule

Two stops.
Zero compromise.

/* Only two keyframe stops — 0% and 100% */ @keyframes anim-opacity { 0% { opacity: 0; } 100% { opacity: 1; } } @keyframes anim-scale { 0% { transform: scale(0.1); } 100% { transform: scale(1.4); } } @keyframes anim-move { 0% { transform: translateX(-80px); } 100% { transform: translateX(80px); } } @keyframes anim-rotate { 0% { transform: rotate(0deg); } 100% { transform: rotate(360deg); } } @keyframes anim-skew { 0% { transform: skewX(-30deg) skewY(-5deg); } 100% { transform: skewX(30deg) skewY(5deg); } } @keyframes anim-combined { 0% { transform: translateY(40px) rotate(0deg) scale(0.5) skewX(0deg); opacity: 0.2; } 100% { transform: translateY(-40px) rotate(180deg) scale(1.2) skewX(15deg); opacity: 1; } }
